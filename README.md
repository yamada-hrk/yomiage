# 小説読み上げ (VOICEVOX)

[VOICEVOX](https://voicevox.hiroshiba.jp/) を使って**小説家になろう**と**カクヨム**の小説を読み上げるChrome拡張機能です。

## 機能

- 小説家になろう・カクヨムの小説ページを自動読み上げ
- 読み上げ中の段落をハイライト表示・自動スクロール
- 段落をクリック/タップしてその位置から再生開始
- スピーカー・読み上げ速度の変更
- 一時停止 / 再開 / 停止
- スペースキーで一時停止/再開トグル
- 読み終わったら自動で次のページへ移動
- 次ページ移動後も自動で読み上げを再開
- 先読み機能（指定段落数分を事前に合成してバッファリング）
- VOICEVOXのホストアドレスを変更可能（リモート接続対応）

## 必要なもの

- Google Chrome（またはChromiumベースのブラウザ）
- [VOICEVOX](https://voicevox.hiroshiba.jp/) がローカルまたはネットワーク上で起動していること

## インストール

1. このリポジトリをクローンまたはZIPでダウンロードする
2. Chromeで `chrome://extensions` を開く
3. 右上の「デベロッパーモード」をオンにする
4. 「パッケージ化されていない拡張機能を読み込む」をクリックし、`extension` フォルダを選択する

## 使い方

1. VOICEVOXを起動する（デフォルトポート: `50021`）
2. 小説家になろうまたはカクヨムの小説本文ページを開く
3. ツールバーの拡張アイコンをクリックしてポップアップを開く
4. VOICEVOXへの接続が確認されたら「再生」ボタンを押す

段落をクリック/タップするとその段落から読み上げを開始できます。

## 設定

| 設定項目 | 説明 |
|---|---|
| VOICEVOXホスト | 接続先のアドレスとポート（例: `localhost:50021`） |
| スピーカー | VOICEVOXのキャラクター・スタイルを選択 |
| 読み上げ速度 | 0.5x〜2.0x の範囲で調整 |
| 先読み段落数 | 再生中に次の0〜5段落分を事前合成してバッファリング。APIの遅延対策に有効 |
| 読み終わったら次のページへ | 有効にすると最終段落の読み上げ後に次話へ自動移動 |
| 遷移まで | 次ページへ移動するまでのカウントダウン秒数（3 / 5 / 10秒） |

## キーボードショートカット

| キー | 動作 |
|---|---|
| `Space` | 一時停止 / 再開 |
| `Escape` | 次ページへの自動遷移をキャンセル |

## 対応サイト

| サイト | 対応URL |
|---|---|
| 小説家になろう | `https://ncode.syosetu.com/*/` |
| カクヨム | `https://kakuyomu.jp/works/*/episodes/*` |

## アーキテクチャ

```
popup.js ──メッセージ──► content.js（各タブに注入）
popup.js ──メッセージ──► background.js（Service Worker）
background.js ──HTTP──► VOICEVOX API
```

- **background.js** — VOICEVOXへのHTTPリクエストを代理。`/audio_query` でクエリ生成後、`/synthesis` でWAV音声を取得しBase64に変換してcontent.jsへ返す
- **content.js** — 小説テキストの抽出・ハイライト・音声再生・先読みキャッシュを管理するメインロジック
- **popup.js** — 設定の管理とボタン操作のUI
